https://nuxtjs.org/

https://vuex.vuejs.org/
* Charts.js
https://www.chartjs.org/
* API
https://covid19api.com/

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
